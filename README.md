## GoodGhosting API

#### Install and Run locally

```bash
$ npm install
$ cp .env.example .env
```
Set NODE_URL in .env file. You can use this one for test purpose.
```
NODE_URL=https://speedy-nodes-nyc.moralis.io/ff176f71021adb540fbb5fa0/eth/kovan
```

```bash
$ npm start
```

#### Run tests

```bash
$ npm test
```
